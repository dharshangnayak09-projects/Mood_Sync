# MoodSync 🎧

MoodSync is an emotion-aware web application that detects user mood in real time and dynamically recommends music to match or improve emotional state.

---

## 🚀 Problem Statement

Users often experience fluctuating emotions such as stress, fatigue, or low mood during daily activities, which impacts productivity and well-being. Existing music platforms provide static recommendations and do not adapt to real-time emotional states.

MoodSync solves this by creating an emotion-aware system that dynamically adapts music recommendations and UI based on the user's current mood.

---

## 💡 Solution

MoodSync uses real-time emotion detection to:
- Identify the user's emotional state
- Recommend music accordingly
- Allow users to either **match** or **change** their mood
- Provide an adaptive and interactive user experience

---

## ✨ Features

- 🎥 Real-time emotion detection (AI-based)
- 🎵 Smart music recommendation system
- 🔁 Match Mood / Change Mood modes
- 🎧 In-app music player
- 🎨 Dynamic UI based on emotion
- ⚡ Fast and responsive web interface

---

## 🧠 How It Works

1. Camera detects user emotion (happy, sad, stressed, etc.)
2. Emotion is sent to backend
3. Backend selects appropriate songs
4. Frontend displays songs and plays them inside the app
5. UI adapts based on emotion
